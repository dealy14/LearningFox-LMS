Flash AS2 Demonstration

This is a simple demonstration of the pipwerks SCORM API wrapper class (ActionScript 2) used in conjunction with the pipwerks SCORM API JavaScript wrapper.

This demonstration requires Flash Player version 8 or higher.

This example created by Philip Hutchison, February 2008. http://pipwerks.com/lab/scorm
