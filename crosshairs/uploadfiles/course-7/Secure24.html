IT Resources - E-mail Attachments

Do not open suspicious e-mail attachments. The Melissa virus spread because it came in an e-mail from a familiar address. The Kama Sutra worm spread by enticing users to open up the e-mail and attachment.

If you must open an attachment before you can verify the source, we suggest the following procedure:

  • Be sure your virus definitions are up-to-date.
  • Save the file to your hard disk.
  • Scan the file using your antivirus software.
  • If the file is virus-free, then open the file.
  • If the file contains a virus, delete it and notify the source.
