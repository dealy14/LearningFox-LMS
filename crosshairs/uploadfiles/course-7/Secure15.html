Countermeasure: Password Protected Screensavers - How?

How do you set up password protection?

If you use a computer that is running Windows XP, your workstation should be automatically set to turn on the password protected screensaver after 15 minutes of inactivity. Your password will be your workstation password.

To lock down your computer yourself you can either:

  • Press the Windows key + the "L" key (simultaneously).
    OR
  • Press CTRL+ALT+DEL (simultaneously) and select “Lock Computer” in the Windows Security window.
