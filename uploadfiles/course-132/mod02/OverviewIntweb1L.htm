Overview - International Web Version

 

Move your mouse cursor over any area on International Web version window below to read descriptions of each area's function.

Type - Indicate the type of issue by selecting Powertrain or NonPowertrain. Powertrain issues involve Base Engine, Drivability, Trans/T-Case, and/or Axles/Final Drive. Select NonPowertrain for all other issues.

Category - Indicate the issue category by selecting Pre-Delivery Quality or Reliability. Select Pre-Delivery Quality if the vehicle was delivered to the dealer, but is not in the customer's possession. Select Reliability if the vehicle has been in the customer's possession.

LOP / TSB - Select the Labor Operations group or Technical Service Bulletin from the drop-down list.

Affected Vehicle - Under Affected Vehicles, enter the last 8-digits of the VIN. Only the last 8-digits are required for this application. Of the last 8 digits, the first two digits of the VIN represent the model year and plant code and the last six digits represent unique vehicle numbers. Enter the Mileage and select the Dealer Code.

Parts - If the part is available for return, select the Part Available box and enter the Part Number, Quantity, and Description.

Customer Concern - Type the customer concern here. Refer to Report Issue for more information on the proper way to report a new issue.

Repair Action - Type the repair action here and indicate whether the repair was successful. Refer to Report Issue for more information on the proper way to report a new issue.

Images - Click the Browse button here to search for and attach images to the report.

Hold - Click this button to save the report, without sending it.

Submit - Click this button to submit or send the report.

Reset - Click this button to clear all fields and start again.
