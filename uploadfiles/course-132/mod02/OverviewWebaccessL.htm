Overview - Standalone Web Version  

Move your mouse cursor over any area on Standalone Web version window below to read descriptions of each area's function.

Issue Type - Indicate the type of issue by selecting Powertrain or NonPowertrain. Powertrain issues involve Base Engine, Drivability, Trans/T-Case, and/or Axles/Final Drive. Select NonPowertrain for all other issues.

Issues Category - Indicate the issue category by selecting Pre-Delivery Quality or Reliability. Select Pre-Delivery Quality if the vehicle was delivered to the dealer, but is not in the customer's possession. Select Reliability if the vehicle has been in the customer's possession.

Root Cause - Type the root cause here. Refer to Report Issue for more information on the proper way to report a new issue.

Repair Action - Type the repair action here. Refer to Report Issue for more information on the proper way to report a new issue.

Effected Vehicle - Under Effected Vehicles either select the vehicle from the drop-down list or enter the vehicle information. Enter the last 8-digits of the VIN. Only the last 8-digits are required for this application. Of the last 8 digits, the first two digits of the VIN represent the model year and plant code and the last six digits represent unique vehicle numbers. Enter the Mileage, Dealer Code, and indicated whether parts were replaced.

Parts - Click this button to complete the parts return form.

Customer Concern - Type the customer concern here. Refer to Report Issue for more information on the proper way to report a new issue.

Images - Click this button here to attach images to the report.

Powertrain Options - If you selected Powertrain as the Issue Type, these Powertrain options will become active. Click to select the options that apply to the concern being reported.

Resend - Click this button to resend the report when changes have been made.

Delete - Click this button to delete the report.

Hold - Click this button to save this report, without sending it.

Close - Click this button to close the issue report window.

Delete - Checking this box deletes the VIN you entered.

Save - Click this button to save the VIN, mileage, and dealer information you just entered.
