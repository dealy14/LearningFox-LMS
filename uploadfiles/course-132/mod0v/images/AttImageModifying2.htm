Modifying Images :continued

  

If the version of CAGRIS you're using does not have mark-up tools or if you need to use more advanced mark-up capabilities, you can modify the image using another software application prior to attaching the image to the report.

Many times, the software that comes with your digital camera has mark-up capabilities. Also, there are other graphics programs such as Image Composer, PaintShop Pro, or Photoshop that have the ability to modify digital images.
