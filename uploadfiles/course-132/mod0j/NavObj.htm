Welcome - Course Objectives  

After completing this course, you will be able to:

  • Navigate this course using the navigation and menu bars
  • Submit a CAGRIS report using one of the CAGRIS reporting versions
  • Properly report a new issue in CAGRIS, entering the appropriate detailed information
  • Attach a properly modified image to a CAGRIS issue report
  • Properly complete the parts return form and input the appropriate information
  • Understand and use the Quality Narrative Analyzer (QNA) and Parts Return Analysis System (PRAS)
