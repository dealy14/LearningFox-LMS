Overview - Standalone Access Version  

Move your mouse cursor over any area on Standalone Access version window below to read descriptions of each area's function.

Type - Indicate the type of issue by selecting Powertrain or NonPowertrain. Powertrain issues involve Base Engine, Drivability, Trans/T-Case, and/or Axles/Final Drive. Select NonPowertrain for all other issues.

Category - Indicate the issue category by selecting Pre-Delivery Quality or Reliability. Select Pre-Delivery Quality if the vehicle was delivered to the dealer, but is not in the customer's possession. Select Reliability if the vehicle has been in the customer's possession.

Zone - Select your zone here.

Affected Vehicle - Under Affected Vehicles, enter the last 8-digits of the VIN. Only the last 8-digits are required for this application. Of the last 8 digits, the first two digits of the VIN represent the model year and plant code and the last six digits represent unique vehicle numbers. Enter the Mileage Month, Day, Hour, Build Date, Dealer Code, and select the Parts Replaced box, if applicable.

Parts - Click the Parts button here to enter the Part number, Quantity, Description, and indicate whether the part is available.

Customer Concern - Click Concerns to enter details about the customer's concern. Refer to Report Issue for more information on the proper way to report a new issue.

Model - Select the vehicle model here.

Images - Click the Images button here to search for and attach images to the report.

Powertrain Check Box Option - If you selected Powertrain as the Issue Type, these Powertrain options will become active. Click to select the options that apply to the concern being reported.

Engine - Select the engine type here.

Transmission Type - Select the transmission type here.

Reset - Click this button to clear all fields and start again.

Sales Code - Enter sales codes here.

Next - Click this button to view or edit the next report in the issue directory.

Previous - Click this button to view or edit the previous report in the issue directory.

New - Click this button to create a new issue report.

Close - Click this button to close the issue report window.

Attached image file names appear here.
